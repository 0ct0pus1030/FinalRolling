一个 2D demo，用帧同步保证 Windows 和 Mac 对战结果一致。

- **本地打木桩**：一个人控制，剩下俩发呆
- **局域网对战**：双人连同一台服务器就能打（服务器另外跑）

  移动
- **J** 攻击
- **K** 翻滚
- **L** 格挡

对战模式
1. 确保服务器已经跑起来
2. 连上后自动开局

## 代码结构

- `Core/`：定点数
- `Logic/`：状态机、攻击判定、物理、事件处理
- `Network/`：连服务器的逻辑，部分网络配置设置
- `View/`：把逻辑数据转成动画和位置

攻击/受击范围，伤害都在SO文件里配。
